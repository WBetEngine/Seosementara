Untuk deploy Cloudflare Worker sse-platform (bukan layanan GitHub). Nilai ini sama dengan GitHub Secrets CLOUDFLARE_*.

Buat token di Cloudflare Dashboard Cloudflare

Langkah 2 · Deploy Worker Cloudflare

GitHub PAT dipakai agar GitHub Actions bisa menulis secrets dan menjalankan workflow Deploy Platform Worker — yang mengunggah kode ke Cloudflare.

  1. Isi PAT → klik Simpan PAT & deploy
  2. Tunggu workflow hijau di GitHub Actions
  3. Salin URL https://sse-platform.*.workers.dev → tempel di bagian 2c di bawah
Buat PAT GitHub Secrets GitHub Actions
