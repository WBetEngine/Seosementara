URL tidak diketik manual — muncul setelah deploy worker:
  1. Jalankan Deploy worker (Actions) (isi token + account dari langkah 1).
  2. Buka run hijau → log job → cari deployment-url (mis. https://sse-platform.*.workers.dev).
  3. Atau Workers di Cloudflare → layanan sse-platform → Trigger.

Setelah deploy, URL juga bisa dari ?api=... atau platform-api-url.js (CI).

Urutan: Cloudflare Worker (token + account) → GitHub PAT (simpan secrets + deploy worker) → Zone/DNS → infra. Tombol Test memanggil Platform API nyata setelah worker hidup.
